EXERCICI 4


Per Executar:

cd (ruta on està guardat el programa de java) 

dir -->per comprobar si està realment el programa de java que volem executar, haurem de veure dos tipologies class i java per poder executar

Si no tenim .class haurem de compilar el programa. Per tant serà javac App.java

Tornarem a fer un dir per veure si s'ha compilat correctament.

cd java App.java 


El programa ens demana la ruta on està guardat el fitxer que volem mostrar per pantalla, li indiquem e imprimirà per el cmd el contingut del fitxer.
